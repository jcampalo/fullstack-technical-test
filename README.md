## Instructions

1. Ensure you use Node Dubnium LTS, if you have NVM installed simply run:

```
nvm use
```

2. Install dependencies:

```
npm ci
```

3. Build project:

```
npm run build
```

4. Run project:

```
npm start
```

5. Now you can open the following url in your favorite browser(not IE please) http://localhost:3000/.

6. Enjoy :)
